Robot Demo

Paste the demo script you want the robot to follow below. Press play to play the script and stop to stop the demo. Stop will finalise the current iteration of the script and return to the init position.
Sample: [{"Grips":2,"Wrist":10,"Elbow":32,"Shoulder":30,"Base":50,"Led":1},{"Grips":1,"Wrist":5,"Elbow":32,"Shoulder":30,"Base":50,"Led":1}]

Move the Robot

Put the final position you want the robot to move to.

Moving to:
{"Grips":,"Wrist":,"Elbow":,"Shoulder":,"Base":,"Led":}
RobotDescriptionMovement
Grips 0 (close) - 3 (completely open)
Wrist 0 (completely down) - 20 (completely up)
Elbow 0 (completely down) - 64 (completely up)
Shoulder 0 (completely down) - 60 (completely up)
Base 0 (completely left) - 100 (completely right)
Led On or off
